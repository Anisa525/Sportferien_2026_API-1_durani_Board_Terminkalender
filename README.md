# Sportferien_2026_API-1_durani_Board_Terminkalender

## Projektbeschreibung

Dieses Projekt ist ein **webbasierter Terminkalender**, mit dem Termine wie **Prüfungen, ÜK-Tage, Geburtstage und andere Ereignisse** erfasst und verwaltet werden können.

Die Anwendung besteht aus:

* einem **Frontend** (Benutzeroberfläche im Browser)
* einem **Backend** (Server / API)
* einer **Datenbank**, in der alle Termine gespeichert werden

Ziel ist es, dass eingetragene Termine **persistent gespeichert** werden und auch nach einem Neuladen der Webseite erhalten bleiben.

---

## Projektziele

* Erstellen einer einfachen Webseite zur Terminverwaltung
* Trennung von **Frontend** und **Backend**
* Verwendung von **Node.js** für Backend und Frontend
* Anbindung an eine **Datenbank** (z. B. MySQL)
* CRUD-Funktionalität:

  * Termine erstellen
  * Termine anzeigen
  * Termine bearbeiten
  * Termine löschen
* Kein Multi-User-System (nur ein Benutzer)

---

## Technologien

### Frontend

* HTML
* CSS
* JavaScript
* Node.js (z. B. mit Express oder als statisches Frontend)

### Backend

* Node.js
* Express.js (REST API)

### Datenbank

* MySQL
* MySQL Workbench / MySQL Shell

## Funktionalität

* Termine über ein Formular erfassen (Titel, Datum, Beschreibung, Typ)
* Speicherung der Termine in der Datenbank
* Anzeige aller gespeicherten Termine auf der Webseite
* Bearbeiten und Löschen bestehender Termine

---

## Installation & Setup (Kurzfassung)

1. Node.js installieren
2. Projektordner erstellen
3. Backend und Frontend trennen
4. Abhängigkeiten installieren (z. B. Express, MySQL)
5. Datenbank erstellen und verbinden
6. Server starten
7. Webseite im Browser öffnen
